ABOUT THE CODE

This is a Streamlit-based web application for analyzing user feedback data and performing text classification tasks.
The application utilizes various natural language processing (NLP) models from the Hugging Face Transformers library.

----------------------------------------------------------------------------------------------------------------
HOW TO USE

1)Downloading the Code: Clone this repository to your local machine or download it as a ZIP file.
------
2)Installing Required Libraries: Install the required libraries in your Python environment.
You can do this by running the command "pip install -r requirements.txt."
------
3)Starting the Streamlit App: Navigate to the project directory in your terminal or command prompt
and run the command "streamlit run app.py".
------
4)Using the Text Interface: Once the Streamlit app starts, you will receive a URL in your terminal
Visit this URL in your browser to access the text interface.
------
5)Performing Tasks: The text interface may contain various options or fields.
You can use this interface to perform tasks such as text analysis or classification.
-----------------------------------------------------------------------------------------------------------------
USAGE


Data Analysis: Select the "Main" tab to view user feedback and happiness rates.

Text Classification: Navigate to the "Classification" tab to classify text into predefined categories.

Bug Fixes Analysis: Explore bug fixes and sentiment distribution in the "Bug Fixes" tab.

Major Updates Analysis: Analyze major updates and sentiment distribution in the "Major Updates" tab.
